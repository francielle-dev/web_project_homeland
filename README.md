# Tripleten web_project_homeland
Projeto de treino desenvolvido com HTML e CSS, conta a história de funcionários da Tripleten.
## 📖 Sobre o projeto

O **De Pátria para Pátria** é uma página web estática que conta a história de funcionários e contribui para o crescimento profissional geracional.

Este projeto foi criado como parte do aprendizado inicial em desenvolvimento web, com foco em estruturação com HTML e estilização com CSS.

## 🎯 Objetivo

* Praticar conceitos básicos de HTML
* Aprender estilização com CSS
* Desenvolver a organização de uma página web
* Criar um layout simples e funcional

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

## ▶️ Como visualizar o projeto

Você pode visualizar o projeto de duas formas:

1. Baixando o repositório:

```bash
git clone https://github.com/francielle-dev/web_project_homeland.git 
```

2. Abrindo o arquivo `index.html` no navegador
